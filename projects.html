Projects

Destin

Destin is a project created during the Global AI Hackathon in Toronto. It is a chat-bot that can discuss different locations around the world with the user. It uses Microsoft's API LUIS to analyze the user's text. From that text, it obtains two key elements. One is the intent of the sentence, and the other is the subject of the sentence. I led development in JavaScript to integrate this API in our application and then produce appropriate responses based on those inputs. This was a great experience since I had never programmed in JavaScript before. Our team attained 2nd place in the hackathon.

Chess Engine

I am in the process of developing a chess engine in Java using the tutorial series "Java Chess Engine Tutorial" by Software Architecture & Design. This has been a great experience because it exercises my object-oriented skills in Java. Also, the Minimax algorithm will be used to compute an optimal move given a set of games that have been previously played. This makes the engine improve over time.
